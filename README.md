# 供应商财务健康度分析系统

这是一个基于真实财报数据的供应商财务健康度分析系统，可以实时查询中国A股上市公司的财务指标。

## 功能特点

- ✅ **真实数据**：基于新浪财经公开的真实财报数据
- 🔍 **模糊搜索**：支持按公司名称或股票代码搜索
- 📊 **核心指标**：展示五大财务健康度核心指标
  - 资产负债率
  - 应付账款周转率
  - 利润变现率
  - 短期负债偿还能力（流动比率）
  - 应收账款周转率
- 📈 **详细财报**：包含四大类详细财务信息
  - 现金流风险指标
  - 上下游供应风险指标
  - 盈利能力风险指标
  - 其他风险指标
- 📅 **最新报告期**：自动获取并展示最新的财报数据
- 📋 **本期/上期对比**：清晰展示本期数、上期数和环比变化

## 安装步骤

### 1. 确保已安装 Python 3.8+

```bash
python --version
```

### 2. 安装依赖

在项目目录下运行：

```bash
pip install -r requirements.txt
```

### 3. 启动服务器

```bash
python app.py
```

### 4. 访问系统

在浏览器中打开：
```
http://localhost:5000
```

## 使用方法

1. **搜索公司**
   - 在搜索框中输入公司名称（如"比亚迪"）或股票代码（如"002594"）
   - 系统会自动显示匹配的公司列表

2. **查看财务数据**
   - 点击搜索结果中的公司
   - 系统会自动加载最新的财报数据
   - 核心指标以卡片形式展示，便于快速查看
   - 详细财务信息以表格形式展示，包含本期数、上期数和环比

3. **数据说明**
   - 数据来源：新浪财经公开API（通过AkShare库）
   - 更新频率：跟随上市公司财报发布
   - 报告期：默认显示最新的报告期（季报/半年报/年报）
   - 缺失数据：部分公司某些指标可能缺失，会标注为"缺失"

## 技术栈

### 后端
- **Flask**：Python Web框架
- **AkShare**：财经数据接口库，获取真实财报数据
- **Pandas**：数据处理

### 前端
- **原生 JavaScript**：无依赖，轻量级
- **Fetch API**：异步数据请求
- **CSS Grid/Flexbox**：响应式布局

## 注意事项

1. **数据准确性**：数据来源于新浪财经公开API，请以上市公司官方公告为准
2. **网络连接**：需要互联网连接以获取实时数据
3. **数据延迟**：财报数据会有一定延迟，通常在公司发布财报后1-2天内更新
4. **CORS设置**：已配置CORS，支持跨域访问

## 常见问题

### Q1: 为什么有些公司搜索不到？
A: AkShare只支持A股上市公司，港股、美股暂不支持。

### Q2: 为什么某些指标显示"缺失"？
A: 部分公司的财报中可能未披露某些详细项目，或者报告期不同导致数据格式差异。

### Q3: 数据多久更新一次？
A: 数据实时从新浪财经获取，跟随上市公司财报发布周期更新（季报、半年报、年报）。

### Q4: 可以导出数据吗？
A: 当前版本不支持导出，可以手动复制或截图保存。
